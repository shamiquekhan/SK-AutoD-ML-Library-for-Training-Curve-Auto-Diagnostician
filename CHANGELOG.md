## [0.1.0] - 2026-05-11

### Added
- Core 10 detectors: OverfittingDetector, ExplodingGradientDetector, LRHighDetector, LRLowDetector, UnderfittingDetector, DyingReLUDetector, NoisyTrainingDetector, DataLeakageDetector, EarlyStopDetector, LabelNoiseDetector
- Text formatter and CLI entry point (`sk-autod diagnose`)
- JSON and HTML output formatters
- Unit test suite with pytest
- Pure Python implementation (no external dependencies for core library)

### Changed
- Initial public release
